ROOMORA

Roomora is a web application that helps students find rooms and compatible roommates using smart matching, chat, and location-based search.


FEATURES
- User authentication (student / owner)
- AI-based roommate matching
- Room listings with filters
- Image & 360° view support
- Real-time chat (Socket.IO)
- Google Maps integration
- Save/bookmark rooms


DATABASE
- Users: basic user info
- MatchProfiles: preferences for matching
- Rooms: room listings
- RoomImages: multiple images
- Messages: chat system
- SavedRooms: bookmarked rooms


TECH STACK
- Node.js, Express
- MySQL, Sequelize
- JWT Auth
- Socket.IO
- Google Maps API


MODULES
- Auth & Matching
- Rooms & Listings
- Chat & Integration
